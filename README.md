# EBidoul | Future of AI

This is the personal portfolio website for **EBidoul**, hosted on GitHub Pages.

## Project Overview

The site has been completely overhauled to feature a **Modern AI / Futuristic** aesthetic. It serves as a digital playground and portfolio to showcase projects at the intersection of Artificial Intelligence and creative coding.

### Key Features
*   **Modern Dark Theme**: A deep slate background with vibrant neon gradients (cyan, purple, pink).
*   **Glassmorphism**: Frosted glass effects on cards and navigation elements.
*   **Responsive Design**: Fully responsive layout with a sticky navigation bar.
*   **Smooth Animations**: Entrance animations for text and a floating effect for the hero image.
*   **Typography**: Uses the modern 'Outfit' font family from Google Fonts.

## Tech Stack
*   **HTML5**: Semantic structure.
*   **CSS3**: Custom variables, flexbox, gradients, and keyframe animations.
*   **GitHub Pages**: Hosting.
